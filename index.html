Project Overview

Create an interactive to-do list that allows users to add, edit, delete, and reorder tasks using drag-and-drop functionality. The project will use HTML, CSS, and JavaScript.

Functionality/features

  • Add Tasks – Users can input a task and add it to the list.
  • Edit Tasks – Users can modify existing tasks.
  • Delete Tasks – Tasks can be removed individually.
  • Drag and Drop Reordering – Users can rearrange tasks by dragging them.
  • Task Categories (Optional) – Users can organize tasks under different sections (e.g., Work, important to least important, Personal).
  • Animations & Styling – Use CSS transitions to make interactions smooth.

    • Usage of programming languages:

    • HTML – Structure of the to-do list.
    • CSS – Styling for a clean UI, hover effects, and transitions.
    • JavaScript – Logic for adding, deleting, editing, and dragging tasks.

      • Possible future features

      • Deadline Reminders – Show alerts for overdue tasks.
      • Progress Bar – Display completion percentage.
